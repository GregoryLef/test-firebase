## ENGLISH
This project is a simple pizza menu application built with Next.js and Firebase (Firestore).
The main goal was to get a first hands-on experience with Firebase by following a tutorial and completing exercises.

What I learned
- Setting up a Firebase project and connecting it to a Next.js app
- Fetching data from Firestore with getDocs


## Français
Ce projet est une application de menu pizza réalisée avec Next.js et Firebase (Firestore).
L'objectif principal était de découvrir Firebase pour la première fois en suivant un tutoriel et en réalisant des exercices pratiques.

Ce que j'ai appris
- Configurer un projet Firebase et le connecter à une application Next.js
- Récupérer des données depuis Firestore avec getDocs

Tech Stack
- Next.js / React
- TypeScript
- Firebase / Firestore
- Tailwind CSS
